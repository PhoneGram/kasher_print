# الكاشير - نظام إدارة المطابع

نظام إدارة متكامل للمطابع، يعمل على المتصفح ويمكن تثبيته كتطبيق على الجوال.

## 📋 المميزات

- 📝 إنشاء وإدارة الفواتير
- 👥 إدارة الزبائن
- 📦 تتبع المخزون والأوراق المتوفرة
- 💰 محفظة لإدارة الأرصدة (الحساب الرئيسي والعامل)
- 📊 إحصائيات يومية وشهرية وسنوية
- 🌙 الوضع الداكن والفاتح
- 📱 تصميم متجاوب مع جميع أحجام الشاشات
- 📸 تحميل الفواتير والجداول كصور PNG بحجم A4
- 📲 قابل للتثبيت كتطبيق PWA
- 🌐 يعمل بدون اتصال بالإنترنت
- 💾 تخزين محلي كامل (localStorage)
- 📥 تصدير واستيراد البيانات بصيغة JSON
